# Changelog

## 0.1.0 - 2026-06-26

### Added
- 节目单「按场地」视图与活动详情卡：点击活动查看完整信息（日期/时间/时长/场地/类别/简介），点击场地查看该场地的全部排期。
- 候鸟沙城地图场地名牌可点击：点击场馆标签直接打开该场地的活动排期，并与侧边栏联动。
- 居民详情整合进「状态」标签：点击「居民动态」中的 AI 居民可查看其聊天与资料，并将相机聚焦到其位置（Focus）。

### Changed
- 顶部招牌精简为「候鸟沙城」单一标题；logo 与浏览器标签页 favicon 统一为「落日 · 沙丘 · 候鸟」。
- 侧边栏标签重排为 状态 / 广播 / 节目单，移除独立的「角色」标签（并入「状态」）。
- 「广播」说明栏改为固定顶栏，消除滚动时的悬浮违和。
