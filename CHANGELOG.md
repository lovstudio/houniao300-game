# Changelog

## 0.5.2 - 2026-06-28

### Added
- 走近建筑即可进入内景；当附近有多个可进入目标时，弹出「附近列表」消歧供选择。

### Changed
- 空间/作品/活动统一复用同一套内景进入组件，进入流程接入生成几何。

## 0.5.1 - 2026-06-28

### Added
- `kind='space'` 的作品支持「走进这座建筑」，进入对应的可走动内场（复用内场独立世界）。

## 0.5.0 - 2026-06-28

### Added
- 内场升级为独立世界：内场地图成为真正的共享 world（多人实时同步），支持碰撞、WASD 移动与侧栏布局；含独立 world 的后端基建。

## 0.4.0 - 2026-06-28

### Added
- 四角色体系（游客/艺术家/志愿者/管理员）+ 邀请码：管理员在设置里分发动态邀请码、按角色分工能力；复制邀请码改为复制完整邀请文案（身份/用法/权益）。
- @AI 上下文对话：主视图底部「传话器」（语音/文字 + @ 角色），@AI 基于候鸟300知识库 RAG 作答，并接入角色长期记忆（memory 向量检索）。
- 通知系统：广播与通知合一为右上角动态浮层，正序时间流、展开默认滚到最新、筛选「全部/我的对话/广播」、未读徽标上限放宽到 99+。
- 移动端：虚拟摇杆移动 + 捏合缩放抑制误导航 + 触屏交互按钮，交互键移到右下避免与摇杆重叠。
- 内场地图支持角色点击走动；撞墙时角色朝墙轻顶作为触觉反馈。
- onboarding 头像改为上传优先，上传后可勾选生成沙之书风格（可重试/选用）。

### Changed
- 落地页/标题屏多轮重设计：CTA 改朱砂活版墨字 + 发丝描边、合并为单行「开启旅程」，整体协调性优化；首屏加载改暖砂风与 onboarding 一致。
- 作品 tab 标题改「作品集」、活动去掉「按场地」视图归并到空间集；侧栏「状态」标签改「系统」、tab 显示完整标签。
- 资产生成仅限作者本人 + 折叠入口，海报改卡片；定位/网络状态从悬浮条移入设置面板。

### Fixed
- 切换操作模式不再连带改动相机跟随（两者解耦）。
- 动态浮层改用不透明暖深色底，修复地图上文字看不清。

## 0.3.1 - 2026-06-28

### Added
- 设置面板底部显示当前版本号（接入已有的 `__APP_VERSION__`，自动跟随 package.json）。

## 0.3.0 - 2026-06-28

### Added
- 空间/作品/活动三类物料体系：空间详情可进入内景、地图定位、查看活动列表与拍照；作品点位支持进入专属体验（上传照片风格化 + 开始游戏），集成 76 件作品图片到点位详情卡；走近空间/作品按空格即可查看详情。
- 照片记忆（photo-memory）：可选位置拍照、相册、点击查看大图、可选提示词生成与按提示词重绘、追问式对话生成、每张追问图都留档、系统风格开关与前端 trace。
- 头像与人物：支持上传真人照片生成沙雕风头像；人物改用程序化简笔画风格 sprite。
- 现场 GPS：实时定位 + 网络信号 HUD、GPS↔地图锚点标定工具（服务器共享）、标定开关收入设置·调试区。
- 「沙之书·沙城手卷」设计系统：浅羊皮纸卷面侧栏、卷头/卷签印章 tab、diegetic 浮木卷轴拉手；落地页两段式（3A 标题屏 → 双列沙化登记）、标题屏题词逐行缓现。
- 本地 BGM 接入与开关；物料管理后台（源图溯源 + AI 重新生成）；调试区重置身份、wipeAgents/seedAgents 只清并重播 AI。

### Changed
- 世界规则：打开即自动入场（移除手动加入/离开），人类上限提至 20，AI 精简为沙城管理员/游客/艺术家三人。
- UI 全屏化：移除顶部导航栏与底部状态栏，世界交互收进游戏内可折叠悬浮控制台。
- 物料重组为 空间/作品/活动，能力下沉到详情卡。

### Fixed
- 地图：碰撞网格精度翻倍至 128×96、修正薄壁屏障取整、最小缩放改 cover 铺满消除黑边/蓝边。
- 布局：手卷/侧栏在 md+ 占据栅格列而非悬浮覆盖，内层列加 min-w-0 防内容溢出被裁。

## 0.2.0 - 2026-06-26

### Added
- 活动专属 AIGC 连环画体验：每个节目单活动拥有独立的游戏。进入后 AI 基于活动背景与沙城风格逐格生成画面，配可选或自定义的问题，回答后推进剧情，5–8 幕收尾，全程可在侧栏回溯。
- 完成活动获得专属勋章；末幕可留下题词、查看同一活动其他人的结局与题词、再玩一次（每次结果不同），或被推荐下一个活动。
- 连环画长图：将整段体验合成为可保存、分享的长图。
- 全局玩家身份：首次进入强制创建身份（名字、性别、头像）；头像可选预置沙城形象或按描述用 AI 生成，存于云端，勋章墙跨用户可见。
- 文生图能力接入（OpenAI 兼容网关 / ZenMux gpt-image-2），叙事导演默认使用高质量模型。

### Changed
- 活动入口移入节目单的活动详情卡；活动「场地」行支持 hover 浮现定位图标，移除冗余的整行定位按钮。

## 0.1.0 - 2026-06-26

### Added
- 节目单「按场地」视图与活动详情卡：点击活动查看完整信息（日期/时间/时长/场地/类别/简介），点击场地查看该场地的全部排期。
- 候鸟沙城地图场地名牌可点击：点击场馆标签直接打开该场地的活动排期，并与侧边栏联动。
- 居民详情整合进「状态」标签：点击「居民动态」中的 AI 居民可查看其聊天与资料，并将相机聚焦到其位置（Focus）。

### Changed
- 顶部招牌精简为「候鸟沙城」单一标题；logo 与浏览器标签页 favicon 统一为「落日 · 沙丘 · 候鸟」。
- 侧边栏标签重排为 状态 / 广播 / 节目单，移除独立的「角色」标签（并入「状态」）。
- 「广播」说明栏改为固定顶栏，消除滚动时的悬浮违和。
